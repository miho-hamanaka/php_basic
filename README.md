# PHP基礎の勉強記録

## ファイルシステムを扱う関数_date:20210209

### fopen
* r - 読み込みモード
* w - 書き込みモード(上書き)
* a -  書き込みモード(追記)

---

### fgets
* 一行取得

### file
* 取得して配列に格納

---

### fclose
* オープンを閉じる

---

### fwrite
* 書き込み処理 (fopenはw or aにする)

---

### fputcsv
* カンマ区切りで書き込みする、引数に配列を指定

### fgetcsv
* 一行のデータで読み込む、配列で返却
